Sample Visual Context

This is an example of HTML visual context. It gives the translator the context needed to produce a high-quality translation first time. With HTML context, the translator sees the translation 'live' in the context view and so can more easily spot continuity or layout problems. Context can be derived from finished content such as web site pages and email marketing HTML, or can be constructed specifically for translation puposes. In the latter case, additional contextual information not intended for final display can be provided to the translator as part of the context (in the way we're doing with this message).
